# CodeAlpha_Basic_Network_Sniffer
Basic Network Sniffer 🛡️This project is a part of my Cyber Security Internship at CodeAlpha. It is a Python-based tool designed to capture and analyze network traffic in real-time.📝 Project OverviewThe Basic Network Sniffer is a tool that monitors data packets flowing through a network interface. It helps in understanding network protocols, data structures, and how information is exchanged between devices.✨ FeaturesPacket Capture: Monitors live network traffic using the scapy library.Protocol Identification: Detects and displays protocols such as TCP, UDP, and others.IP Tracking: Extracts and shows both Source and Destination IP addresses.Payload Analysis: Displays a hex representation of the data (payload) contained within the packets.🛠️ Technologies UsedLanguage: Python 🐍Library: Scapy 🚀 How to RunClone the repository: ```bashgit clone https://www.google.com/search?q=https://github.com/YOUR_USERNAME/CodeAlpha_Basic_Network_Sniffer.gitInstall dependencies: ```bashpip install scapyRun the sniffer (Administrative privileges required): * Windows: Run your terminal as Administrator.Linux: Use sudo python sniffer.py.⚠️ DisclaimerThis tool is developed for educational purposes only as part of the CodeAlpha internship tasks.
