# Basic Network Sniffer 🛡️
This project is a part of my Cyber Security Internship at CodeAlpha. It is a Python-based tool designed to capture and analyze network traffic in real-time.
# 📝 Project Overview
The Basic Network Sniffer is a tool that monitors data packets flowing through a network interface. It helps in understanding network protocols, data structures, and how information is exchanged between devices.
# ✨ FeaturesPacket Capture: Monitors live network traffic using the scapy library.Protocol Identification: Detects and displays protocols such as TCP, UDP, and others.IP Tracking: Extracts and shows both Source and Destination IP addresses.Payload Analysis: Displays a hex representation of the data (payload) contained within the packets.
# 🛠️ Technologies Used
Language: Python 
Library: Scapy
